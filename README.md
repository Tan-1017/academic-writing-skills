# academic-writing-skills
A curated collection of AI-ready prompt templates for writing each section of a top-tier machine learning conference paper — from abstract to conclusion, with built-in citation integrity.

# 学术写作技能库

本仓库提供一套面向顶级人工智能会议（如 NeurIPS、ICML、ICLR）的英文学术写作技能（Skills）。每个技能均为可直接使用的 AI 提示词模板，能够帮助研究者高效撰写论文的各个章节，同时保证语言地道、逻辑严谨并符合学术规范。

## 技能列表

### 1. 摘要编写
将论文的核心要素（背景、问题、方法、结果、意义）整合为一段符合顶会标准的英文摘要。严格遵循四段式逻辑，信息密度高且无冗余。

### 2. 引言编写
根据用户提供的研究背景、现有问题、方法思想和贡献点，生成逻辑严密、漏斗形推进的英文引言。自动构建“宽泛背景→具体缺口→本文方法→贡献”的叙事链。

### 3. 相关工作编写
将文献按主题分组，撰写流畅的对比式综述段落。强制要求引用近五年顶级会议/期刊的真实论文，并生成对应的 BibTeX 条目，绝不允许虚构文献。

### 4. 方法论编写
将用户的技术草稿、公式和模型描述转化为结构清晰、符号统一的英文学术方法章节。遵循“问题定义→总览→模块展开→训练细节”的递进逻辑，公式与文字紧密配合。

### 5. 实验部分编写
组织实验设置、比较方法、主要结果、消融分析等子章节，以客观、可复现的方式呈现实验证据。自动检查数据一致性，并确保所有结论均有数字支撑。

### 6. 结果与分析编写
超越表层数据，撰写深入的机制解释与讨论。严格遵循“现象回顾→机制解释→与前人对话→局限性→展望”的逻辑，将实验结果提炼为有洞察力的学术讨论。

### 7. 结论编写
以全新措辞重述核心贡献，点明研究意义，并诚实指出局限与具体未来方向。拒绝空洞套话，收尾自然庄重。

### 8. 文献引用插入与 BibTeX 生成
扫描 LaTeX 全文，识别所有需要引用的位置，自动插入真实、权威、近期的文献引用，同时生成规范的 BibTeX 文件。以学术诚信为底线，绝不编造任何文献。

## 使用方法

每个技能均为一个独立的提示词文件（位于 `skills/` 目录下）。使用时，只需将对应文件的内容复制到对话中，再附上你的草稿或关键信息，即可获得高质量的输出。

**典型流程：**
1. 打开你需要的技能文件（如 `abstract.md`）。
2. 将全文复制到支持长上下文的大语言模型（如 Claude、GPT-4）中。
3. 在提示词的末尾 `# Input` 部分粘贴你的中文草稿或要点。
4. 等待模型生成，即可得到可直接粘贴进 LaTeX 源文件的英文段落及中文直译核对版本。

## 技能设计原则

- **零虚构文献**：所有引用均基于真实存在的论文，绝不允许生成虚假或推测的引文。
- **去 AI 味**：提示词内置了大量约束（如禁用破折号、禁止机械连接词、拒绝宣传性词汇），使输出读起来像资深学者亲笔。
- **双栏输出**：每个技能均输出 `Part 1 [LaTeX]` 和 `Part 2 [Translation]`，方便作者直接使用并核验原意。
- **适用顶会**：语言风格和结构要求参照 NeurIPS、ICML、ICLR 等会议的标准。

## 注意事项

- 这些技能仅提供写作辅助，研究者仍需对最终内容的科学准确性负责。
- 生成所有引用文献时，请务必在文献数据库中核实其真实性和准确性。
- 不同大语言模型对提示词的响应可能有细微差异，建议先在少量样例上测试，必要时微调输入。
